Stream Tool
The Stream Tool makes it possible to set up a tournament stream in no time, meant to be used alongside streaming software like OBS. Written in pure Javascript and using Electron for the GUI, the Stream Tool is available for Rivals of Aether, where it's being constantly updated, and it's also available for other games like Super Smash Bros. Melee and Rushdown Revolt.
GUI
The GUI manages what's about to be displayed on stream. With it, you can easily set up player names, tags, scores, characters and their skins, round, casters, and more. Features include: Player presets, Best of 3/5 modes, [W]/[L] indicators for Grand Finals, 2v2 support, and various optional settings.
SCOREBOARD
The Scoreboard is shown on top of the tournament gameplay. In it, we can display the player names and tags, as well as the score, character and current round. Includes support for [W]/[L] indicators, color slot changes, and 2v2 games!
VS SCREEN
The VS Screen is meant to be shown whenever we are waiting for the next match. In it, we can clearly see the players that are playing now or are about to start playing, as well as round, tournament and commentator info.
RoA Skin Recolorer
This web app uses a code used to share custom color skins from the game Rivals of Aether, and generates a transparent image that you can download and use wherever you want. It can also randomize said code to generate a custom skin, and you can also upload your own images to recolor them. You can try it out here, no download required!
